# BookMarks
BookMarks is an iOS application for people who love reading.

The application is build using Cocoapods.
Dependencies:
	- Firebase
	- Cosmos

Supporting real oAuth integration with Firebase (v3)

Databases:

Users
	- email
	- password
	- readBooks
	- toReadBoooks

Books
	- name
	- author
	- description
	- image (string - https URL)
