# BookMarks
BookMarks is an **iOS** application for people who love reading.


##Dependencies:
Dependency management is done using Cocoapods
- Firebase
- Cosmos

Supporting real Authentication powered by **Firebase (v3)**

## Databases:

**Users**
- email
- password
- readBooks
- toReadBoooks

**Books**
- name
- author
- description
- image (string - https URL)
